CSCE 242: Web Applications

Johnny Hyman

This class is designed to teach the basics of web development, from front-end to back-end. Uses HTML/CSS, client and server side JavaScript with node express and MongoDB.

Assignments

  • Assignment 1 - Basic HTML

    Website used to demonstrate the bare basics of HTML elements, such as headings, tables, links, images, and lists.

  • Assignment 2 - Basic CSS

    Website used to demonstrate the basics of CSS, using divs to appropriately name content, classes and id's to style different elements such as images and tables, as well as changing fonts and colors.

  • Assignment 3 - Page Layout

    Website used to demonstrate using CSS flexbox and a media query to format a pleasing looking webpage based on a given example.

  • Assignment 4 - Recreate CSS Page

    Website used to further demonstrate understanding of CSS flexbox and media query skills by attempting to replicate the format of a more complicated webpage.

  • Assignment 5 - Introduction to JavaScript

    Website used to demonstrate an understanding of basic JavaScript syntax, such as functions and element fetching.

  • Assignment 6 - If Statements

    Website used to demonstrate the usage of if statements and hamburger menus in JavaScript. Also further emphasizes sliders.

  • Assignment 7 - For Loops

    Website used to demonstrate the usage of for loops within JavaScript. Also demonstrates understanding of special functions such as setInterval.

  • Assignment 8 - Arrays

    Wesbite used to demonstrate an understanding and usage of arrays within JavaScript.

  • Assignment 9 - Classes

    Website used to demonstrate an understanding of classes within JavaScript. Also incorporate's W3Schools' modal dialog.

  • Assignment 10 - Parsing JSON

    Website used to demonstrate an understanding of async and await keywords to fetch a JSON file and display it appropriately on a webpage.

  • Assignment 11 - Server Side Hello World

    Website used to show the installation of Node.js and the upload of a website server-side through Render.
