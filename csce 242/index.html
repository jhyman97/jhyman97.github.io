CSCE 242: Web Applications

Johnny Hyman

This class is designed to teach the basics of web development, from front-end to back-end. Uses HTML/CSS, client and server side JavaScript with node express and MongoDB.

Assignments

  • Assignment 1 - Basic HTML

    Website used to demonstrate the bare basics of HTML elements, such as headings, tables, links, images, and lists.

  • Assignment 2 - Basic CSS

    Website used to demonstrate the basics of CSS, using divs to appropriately name content, classes and id's to style different elements such as images and tables, as well as changing fonts and colors.

  • Assignment 3 - Page Layout

    Website used to demonstrate using CSS flexbox and a media query to format a pleasing looking webpage based on a given example.
